# Founder Packet Completeness Checklist v2

All must be true:

- mission is explicit
- priority is explicit
- scope is explicit
- non-goals are explicit
- target users are explicit
- success criteria are explicit
- failure criteria are explicit
- tradeoffs are explicit
- authority boundaries are explicit
- escalation boundaries are explicit
- resources are explicit
- founder attention budget is explicit
- existing assets are explicit
- forbidden carry-forward is explicit
- evidence inputs are explicit
- risk register is explicit
- founder decision log is explicit
- dependency map is explicit
- phase boundaries are explicit
- freeze criteria are explicit
- change control is explicit

If any item is missing:

- packet fails
- BPB may not begin
