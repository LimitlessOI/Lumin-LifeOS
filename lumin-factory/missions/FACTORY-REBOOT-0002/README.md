# FACTORY-REBOOT-0002

## Mission

Convert Segments 2 through 4 into real checked-in runtime artifact packs.

## Covered segments

- Segment 2 — Founder Packet contracts
- Segment 3 — Founder Intent / Adam Filter
- Segment 4 — BPB intake and compilation

## Why this mission exists

These segments were designed but not yet emitted as concrete machine payloads.

This mission fixes that by providing:

- exact artifact payload files
- a BPB blueprint pack that points to them
- acceptance tests
- authority checks
- salvage map
- blocker schema

## Important boundary

This mission still does not make the entire rebuild fully coder-ready end-to-end.

It does:

- remove the biggest upstream artifact gap
- convert doctrine into concrete runtime artifacts

It does not yet:

- create full step-atomic coder packets for every later segment
- certify the whole rebuild as deterministically executable by a low-tier coder
