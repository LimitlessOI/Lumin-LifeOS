# Lumin Factory

Standalone factory runtime cutover from Lumin-LifeOS.

## Quick start

```bash
npm install
npm run check
npm start
```

## Verify

```bash
npm run factory:ci
```

## Structure

- `factory-staging/` — live runtime
- `missions/` — blueprint mission packs
- `scripts/` — operator tooling

Built by blueprint missions FACTORY-REBOOT-0011 through 0019.
