Quickstart

Integrate TokenOS in under 5 minutes

1. Replace your API base URL

POST /api/v1/tokenos/proxy
Authorization: Bearer <your-tokenos-key>
Content-Type: application/json

{
  "provider": "openai",       // openai | anthropic | google
  "model": "gpt-4",
  "messages": [...],          // same as OpenAI format
  "tco_mode": "optimized"     // optimized (default) | direct | ab_test
}

2. Read the response

Same format as OpenAI responses, with an extra tco_metrics field:

{
  "choices": [{ "message": { "content": "..." } }],
  "tco_metrics": {
    "savings_usd": "0.004200",
    "savings_pct": "73.2",
    "quality_score": 91,
    "compression_used": true,
    "failed_over": false,
    "latency_ms": 842
  }
}

3. Check your savings

Visit Dashboard or call GET /api/v1/tokenos/dashboard with your key.
