How this screen works

The Word Keeper routes expose commitments as pending items. This overlay helps a user inspect detector output before confirming or rejecting it.

Confirm Marks the commitment as confirmed and enables reminder scheduling when a deadline exists.
Decline Marks the commitment as declined when the detector surfaced a false positive or unwanted item.
Manual add Creates a commitment from typed text using the same backend extraction flow.
