# 08 — Global vs itemized: would itemizing beat ~$4,900?

**Question (Adam):** WRM-style home-birth midwifery charges a **global fee ~$4,900**. Is **itemizing** (visit-by-visit / component codes) more money? Sherry is an **herbalist**, does **breastfeeding help** (not CLC), office visits **≥1 hour**. Do **not** ask Sherry — research only.

**Honesty:** No payer contract for WRM was available. Numbers below are industry-structure reasoning + published coding policy. Amounts for *this* practice’s allowed rates remain `VERIFY` via ClientCare Allowed Amount / fee schedule.

---

## Short answer

| Strategy | Likely vs ~$4,900 global | Why |
|----------|--------------------------|-----|
| Replace global `59400` with `59426` + `59409`/`59410` + `59430` for the **same full episode** | **Usually not more** — often **same or less** | Payers treat itemized split as the **parts of the package**, not a raise. Many policies: if you bill components, you **cannot** also bill global. |
| Keep global **and** bill **outside-the-package** services | **This is where real upside lives** | Labs, imaging (if billed), NSTs, pregnancy-confirmation visit, complication/high-risk E/Ms, newborn, some lactation/E/M — **not** “extra money for longer routine prenatal.” |
| Bill every hour-long prenatal as high-level E/M **on top of** global | **High deny / takeback risk** | Routine antepartum is **inside** global (~13 visits). Visit length alone does not unbundle the package. |

**THINK (ranking):** For a full-care home-birth midwife, **global + capture excluded services** beats **full itemization of the same episode**. Itemization is mainly for **split care** (transfer, insurance change, delivery elsewhere), not a tip to print more.

## What “global” actually buys (industry standard)

Global vaginal maternity **`59400`** (same provider/group full episode) typically bundles:

- ~**13 routine antepartum visits** (monthly → biweekly → weekly pattern)
- Uncomplicated labor management + vaginal delivery
- Routine postpartum care (often through ~6 weeks; some payer docs say longer — `VERIFY`)

Sources: ACOG/AMA package definition as summarized by coding guides (RubinBrown 2025; CareOregon global guide; UHC Obstetrical Policy; Health Partners / Jefferson maternity bulletin).

**Adam’s ~$4,900** = practice **cash/global sticker** (or billed charge). Insurance **allowed** may be lower. Forever-chase is against **allowed/contract**, not sticker.

## What itemization codes mean (not a raise button)

| Code | Meaning | When used |
|------|---------|-----------|
| `59425` | Antepartum only, 4–6 visits | Split / partial prenatal |
| `59426` | Antepartum only, **7+** visits | Split / partial prenatal |
| E/M each | Antepartum **1–3** visits | Very short prenatal relationship |
| `59409` | Delivery only | Someone else did prenatal/PP |
| `59410` | Delivery + postpartum | No full antepartum by this group |
| `59430` | Postpartum only | Delivery elsewhere |

**Payer pattern (KNOW from policy docs):** When antepartum + delivery + postpartum are paid separately, **global is not also paid**. Itemization ≠ stacking on top of `$4,900`.

### Illustrative math (THINK — not WRM rates)

Suppose a payer’s **global allowed** ≈ what they would pay for full care. Component alloweds are often **fractional** (e.g. antepartum-only + delivery-only + PP-only ≈ global).  

If practice **bills** $4,900 global but **allowed** is $3,200, itemizing does not magically create $4,900 cash — it reshuffles the same pie unless **extra outside services** are added.

**Without WRM allowed amounts we cannot prove $X uplift.** Next system pull: Allowed Amount report for `59400` vs `59426`/`59409`/`59410`/`59430` for major payers (BCBS Anthem etc.).

## Where more money actually is (home-birth midwife)

### A. Outside global — usually separately billable (if documented + medically necessary)

| Lever | Codes / notes | Fit for Sherry-type practice |
|-------|----------------|------------------------------|
| Pregnancy confirmation / initial visit before OB record opens | E/M | Often missed |
| Labs (CBC, GCT, GBS, etc.) | Lab CPTs / panels | If practice bills labs |
| OB ultrasound | `76801`–`76828` family | Only if midwife performs/bills imaging — many midwives **refer** (no bill) |
| NST / antepartum testing | e.g. `59025` | If performed |
| Extra visits for **documented complications / high-risk** beyond routine ~13 | E/M + complication ICD + often modifier **25** when same day as other service | **Not** “visit was 60 minutes so bill extra” |
| Problems **unrelated** to pregnancy | E/M | Rare but real |
| Newborn care / attendance | Separate patient/claim | Easy underbill |
| Postpartum **problem** visits beyond routine package | E/M | Mastitis, feeding failure, etc. |
| Prolonged / complex laceration repair | Payer-specific | `VERIFY` |

Industry guides repeatedly: **ultrasound + testing + complication visits** are the common “left on table” dollars next to global — not re-labeling prenatal as itemized E/M.

### B. Hour-long visits (≥60 min)

| Idea | Verdict |
|------|---------|
| “Each prenatal is 60+ min → bill 99215 × 13 **plus** global” | **No** — routine prenatal is in the package; this is classic unbundling. |
| “60+ min → justifies higher **global** fee in cash packages” | **Business pricing**, not insurance coding. Cash global can be $4,900 regardless of CPT. |
| “60+ min **complication** or lactation problem visit **outside** routine” | Time-based E/M **can** apply when visit is **not** routine antepartum (document total time + counseling). |

**THINK:** Long visits improve **care and documentation** (supports medical necessity / appeals) more than they raise the global CPT itself.

### C. Breastfeeding help (not CLC)

| Path | Notes |
|------|-------|
| Midwife E/M for feeding problem (mother and/or baby) | Common coding path when clinician is QHP midwife; document time/MDM |
| Preventive counseling `99401`–`99404` | Some payers use for ACA preventive lactation — **payer-specific** |
| Lactation-specific HCPCS / CLC taxonomy | Often needs **CLC/IBCLC** enrollment — Sherry **not CLC** → many of these lanes closed |
| Inside global education | Routine breastfeeding counseling in prenatal/PP is **often package-included** — don’t double-bill |

**THINK:** Biggest lactation dollar = **problem visits** with diagnosis (e.g. feeding difficulty) billed as E/M, not “S9443 class” without credentials.

### D. Herbalism / herbology

| Path | Notes |
|------|-------|
| Insurance bill for “herbal consult” | **Almost never** — not a standard covered CPT |
| Clinical counseling inside a midwifery visit | May be part of visit documentation; does not create a separate herb CPT |
| Cash / self-pay herb services | Legitimate revenue **outside** insurance |

Do not invent herb CPT stacking onto maternity claims.

### E. Home birth setting

- POS **12** (home) vs office **11** for prenatals — rates/coverage vary hard by payer.
- Some commercial plans underpay or exclude planned home birth; OON superbills common.
- State Medicaid often has **explicit** home-birth billing guides (e.g. WA HCA) — rules differ from commercial global.
- Global `59400` still used for planned home vaginal when covered; itemization still for split care.

## Scenario comparison (THINK)

Assume full prenatal + home vaginal + postpartum by same midwife; ~13+ hour-long prenatals; breastfeeding advice; herbal counsel; some labs ordered.

| Approach | Insurance revenue shape |
|----------|-------------------------|
| **A. Global only** `$4,900` billed | One claim; chase allowed; miss outside services |
| **B. Itemize same episode** `59426+59410` (or similar) | Still one episode pie; **not** 13× hour E/M |
| **C. Global + outside services** | Global + labs + (if any) NST/US + complication E/Ms + newborn + problem lactation E/Ms |
| **D. Cash global $4,900 + patient submits OON** | Patient reimbursement ≠ practice revenue unless assignment |

**Best default hypothesis for WRM:** optimize **C**, keep **A** as the maternity spine, use **B** only when care is truly split. Prove with Allowed Amount pulls.

## Bottom line for Adam

1. **Itemizing the same full pregnancy is unlikely to beat a solid ~$4,900 global** — and can invite denials if done as “more codes = more money.”  
2. **More money** = capture **everything outside the package** + never leave births unbilled + forever-chase underpays against **allowed**.  
3. **Hour-long visits** support quality/docs; they do **not** by themselves unlock 13× specialist E/M on top of global.  
4. **Lactation without CLC** → midwife E/M for problems; not CLC HCPCS.  
5. **Herbs** → cash / clinical counseling inside visits, not insurance line items.
