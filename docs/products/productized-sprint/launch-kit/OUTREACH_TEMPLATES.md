## Objection handling

| Objection | Response |
|---|---|
| "I can just use ChatGPT." | "You can — and it'll sound like ChatGPT. This is built from *your* stories and voice, then I curate it. That's the difference people can feel." |
| "Does it post for me?" | "Not yet — you or your VA paste them (takes minutes). Auto-posting is coming. Right now you're paying for great content, done for you." |
| "Too expensive." | "One client from consistent content pays for this many times over. Start with the $497 Starter and see." |
| "I don't have time for a call." | "No call needed — send me a 3-minute voice note answering one prompt and I'll build from that." |
| "Will it sound like me?" | "That's the whole point — we pull your real words. If a piece doesn't sound like you, I cut it before you ever see it." |

## Consent note

Only message people you have a real relationship with or who opted in. No scraped lists, no bulk cold
email. This mirrors the Outreach CRM consent doctrine (outbound consent, quiet hours) even though this
service is delivered manually.
