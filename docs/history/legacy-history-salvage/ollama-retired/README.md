## Ollama Retired Archive

Founder directive on 2026-06-30 permanently retired Ollama from the live LifeOS and BuilderOS runtime.

This archive contains files that were previously used for:

- local Ollama model setup
- Railway Ollama deployment experiments
- tunnel/bootstrap scripts
- dedicated Ollama adapter code
- Ollama-only setup and operating docs

These files are preserved for historical salvage only.

They are not part of the active runtime spine.

Archive layout:

- `root-docs/` — former repo-root Ollama setup docs
- `docs/` — former docs/ Ollama setup guides
- `scripts/` — former Ollama utility and tunnel scripts
- `packages/` — former adapter implementation
