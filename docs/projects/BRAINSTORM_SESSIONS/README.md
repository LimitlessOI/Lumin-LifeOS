# BRAINSTORM_SESSIONS (artifact folder)

Store one subfolder per session:

`<program-slug>/<YYYY-MM-DD>_<short-topic>/`

See **`docs/BRAINSTORM_SESSIONS_PROTOCOL.md`** for required files (`00_CHARTER.md` … `50_TRIAGE.md`) and phase order.

**Programs:** use different slugs (e.g. `lifeos`, `site-builder`, `tsos-platform`) so sessions stay legible.
