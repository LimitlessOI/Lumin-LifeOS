# Amendment 19 Project Governance Proof: G118-100 - Documentation Review and Linting

## Governance Guideline G118-100

**Guideline:** All new or updated project documentation files (`docs/projects/**/*.md`) must pass a linter check for formatting and a content review by a designated approver group before being merged into the main branch.
