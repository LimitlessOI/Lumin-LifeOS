# Biofeedback VR System

This repository hosts the microservices for a biofeedback VR therapy system, including services for data ingestion, VR environment, AI orchestration, and therapist dashboard.

## Structure

- `services/`: Contains individual microservices.
- `common/`: Shared configurations and utilities.

## Setup

1. Install dependencies: `npm install`
2. Start services: `npm start`
