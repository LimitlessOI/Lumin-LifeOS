# Jest Configurator Customization Guide

This guide helps you customize Jest configurations generated by the CLI tool.

## Base Configuration

The base configuration provides a simple setup suitable for most projects. You can customize it by editing `jest.config.js` in your project directory.

## Framework-Specific Configurations

### Express

- Includes setup files for initializing Express-specific testing environments.

### Fastify

- Includes setup files for initializing Fastify-specific testing environments.

## Adding Custom Setup

To add custom setup files and configurations, modify the generated `jest.config.js` file as needed.

## Advanced Options

For more advanced configurations, refer to the [Jest documentation](https://jestjs.io/docs/en/configuration.html).
